## 稿件标题
Animated Java 原版模型动画制作系列教程

## 摘要
本文介绍了一套关于 Animated Java（简称AJ）原版模型动画制作的系列教程项目，包括基础教程和进阶教程两部分。项目配套提供了作者制作的中文文档和汉化插件资源，旨在降低 Animated Java 的学习门槛，帮助更多创作者在 Minecraft 原版中实现复杂的模型动画效果。
