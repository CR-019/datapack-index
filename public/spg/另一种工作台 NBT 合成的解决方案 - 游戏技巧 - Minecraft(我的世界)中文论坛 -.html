另一种工作台 NBT 合成的解决方案

前言

之前，我们可以通过添加原版模组专用工作台并使用一套自己的逻辑来进行 NBT 合成。这种做法的局限性是______________。（2 分）

原理

当玩家使用了某一合成配方以后，相关配方会被「解锁」，进而能够触发某一进度。在该进度的奖励函数/战利品表中，我们可以清除掉该配方产生的原有物品，将其替换为我们想要的带有 NBT 的真正结果等。

嗟，来食

配方

首先创建一个配方（如 spgoding:enchanted_gold_block，附魔金块），将该配方的原料设置为你想要制作的 NBT 合成配方的原料（如圆石）。请注意该配方的结果必须是一个原版无法获得的物品（如知识之书）：

  1. data/spgoding/recipes/enchanted_gold_block.json
  2. > =======
  3. {
  4.     "type": "minecraft:crafting_shapeless",
  5.     "ingredients": [
  6.         {
  7.             "item": "minecraft:cobblestone"
  8.         }
  9.     ],
  10.     "result": {
  11.         "item": "minecraft:knowledge_book",
  12.         "count": 1
  13.     }
  14. }
复制代码

进度

现在设置一个当解锁了该配方（如 spgoding:enchanted_gold_block）后触发的进度，并指定触发的函数与战利品表。

  1. data/spgoding/advancements/recipes/enchanted_gold_block.json
  2. > =======
  3. {
  4.     "criteria": {
  5.         "criterion": {
  6.             "trigger": "minecraft:recipe_unlocked",
  7.             "conditions": {
  8.                 "recipe": "spgoding:enchanted_gold_block"
  9.             }
  10.         }
  11.     },
  12.     "rewards": {
  13.         "function": "spgoding:recipes/enchanted_gold_block",
  14.         "loot": [
  15.             "spgoding:recipes/enchanted_gold_block"
  16.         ]
  17.     }
  18. }
复制代码

函数

现在写一下刚刚进度里指定的函数（如 spgoding:recipes/enchanted_gold_block），清除掉原有产物，并撤回配方与进度。

  1. data/spgoding/functions/recipes/enchanted_gold_block.mcfunction
  2. > =======
  3. clear @s minecraft:knowledge_book
  4. recipe take @s spgoding:enchanted_gold_block
  5. advancement revoke @s only spgoding:recipes/enchanted_gold_block
复制代码

战利品表

最后写一下刚刚进度里指定的战利品表（如 spgoding:recipes/enchanted_gold_block），返回该 NBT 配方真正的产物。事实上你也可以完全不用战利品表，而是在上面的函数中使用一条 /give 命令。

  1. data/spgoding/loot_tables/recipes/enchanted_gold_block.json
  2. > =======
  3. {
  4.     "pools": [
  5.         {
  6.             "rolls": 1,
  7.             "entries": [
  8.                 {
  9.                     "type": "minecraft:item",
  10.                     "name": "minecraft:gold_block",
  11.                     "functions": [
  12.                         {
  13.                             "function": "minecraft:set_name",
  14.                             "name": {
  15.                                 "text": "点石成金",
  16.                                 "italic": false,
  17.                                 "color": "gold"
  18.                             }
  19.                         }
  20.                     ]
  21.                 }
  22.             ]
  23.         }
  24.     ]
  25. }
复制代码
