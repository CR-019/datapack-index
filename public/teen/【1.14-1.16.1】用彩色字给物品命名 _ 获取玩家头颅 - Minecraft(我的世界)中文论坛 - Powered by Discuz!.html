Name it

用彩色字给物品命名吧！

物品命名

打开书与笔，在第一页写上 {"text":"哈哈哈","color":"red"}，然后点击完成。（不是著名！）

将一块腐肉丢到铁砧上，再将刚才的书与笔也丢到铁砧上，随着一声清脆的敲击声，腐肉就命名完成了。

腐肉的名字变成了 哈哈哈 。

什么？你说你不会写 json？没关系，用小节符号也是可以的，转换成万国码（即 \u00A7）就行（

在书与笔内写上 "\u00A7r\u00A72233"（包括两头的双引号，即 "§r§2233"），再重复上述步骤，你就会得好一块名字为 233 的腐肉。

当然粗体、斜体、下划线、删除线等其他样式也是支持的（

获取头颅

将凋零骷髅头颅和写有玩家 Java 版正版 ID（如 ljyys）的书与笔丢到工作台上（注意先后顺序），然后你就可以获得他 / 她的头啦！

使用命令

命令功能
scoreboard players set #name name_it-value 1或0开启或关闭 物品命名 功能
scoreboard players set #head name_it-value 1或0开启或关闭 获取头颅 功能
function name_it:remove移除模组建立的记分板
