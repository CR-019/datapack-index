1.12，进度系统取缔了原先的成就系统。

1.16，自定义维度进入了人们的视野。

1.17，物品修饰器诞生。

特性

  • 简单好用！功能强大！
  • 支持数十种数据包 JSON 文件格式！
  • 支持所有参数！（如果有遗漏，可以到作者的 仓库 反馈，也可以直接回复在本帖下面）
  • 可以把写好的 JSON 文件通过链接分享给他人！（对中国大陆用户可能不太友好）
  • 可以设置输出 JSON 的缩进样式（2 空格、4 空格、Tab 等）！
  • 可以直接修改右边的 JSON，改动会反应到左侧图形化界面中！
  • 网站页面支持亮色主题与暗色主题！
