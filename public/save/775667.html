简介
该数据包内含根据各种类型/特性进行归类的标签，可作为前置供各位使用。

标签类型可以简单概括为如下类别
  • 相同类型的方块/物品/实体
  • 相同状态的方块

安装步骤
  • 进入对应版本的下载页后，展开Assets，点击Source code (zip) 进行下载
  • 解压下载后的文件UIN-<版本>.zip
  • 确保解压后的文件夹打开后存在data文件夹，pack.mcmeta文件，pack.png图片和LICENSE文件，和一些奇怪的文件
  • 把该文件夹放入存档中的datapacks文件夹内
*若以上步骤于存档打开时执行，游戏内执行命令/reload以重载数据包

MISC.
  • 包内的方块/实体/物品文件夹下各有一份txt格式，包含所有id且不引用任何标签的文本，可供玩家单独使用
  • 该包引用了大量原版标签，因此尽量避免修改原版标签
  • 部分标签仅在新版本添加，而旧版本将不在进行更新
  • 完整的标签列表可打开上方对应版本的页面进行查看
  • 若以上内容与实际情况有出入，或存在漏标。请联系我以便即时修正，感谢
  • 若有其他标签请求可在论坛内回复/私信本人，适用性强的标签、技术性相关的标签优先
  • 过于偏门的标签可能不会被收入，望谅解
  • 只好写点不需要动脑的小玩意来勉强维持生计

感谢您的反馈，illager我收了。

猪灵幼年为被动型，成年则默认为敌对型，玩家持有金装则表现为中立。当初考虑到其特殊性，便根据其默认状态和与wiki同步这两点为由归类至敌对生物。
同样的问题也存在于北极熊(幼年/成年/护崽)、铁傀儡(绝对和平/自然中立)、狼(野生/驯服)和熊猫(默认被动/好斗中立)。

为了尽量满足玩家需求的同时又不过度定义生物的状态，相关标签将进行最低限度的定义，之后将移除此类生物，玩家可以通过创建实体标签，引用现有标签，并定义上合适于他们制作项目的实体类型。至于age可能是我看错了...
