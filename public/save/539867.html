  2. /stats entity @p set AffectedBlocks @p a

  2. /execute @p ~ ~ ~ fill ~-1 ~-1 ~-1 ~1 ~-1 ~1 minecraft:stone 0 replace minecraft:barrier 0
复制代码
*方块操作
  1. /testfor @p[score_a_min=4]
  2. /testfor @p[score_a=6]
