#### 点击事件(1.21.5+) ["click_event"](https://zh.minecraft.wiki/w/?curid=49233#%E7%82%B9%E5%87%BB%E4%BA%8B%E4%BB%B6)
  |     **action**      |     描述     |     **附加标签名**      |     **附加标签内容**      |       可用性       |
  | :-----------------: | :----------: | :----------------: | :----------------: | :----------------: |
  |     `"open_url"`      |   打开网页   |      url     | url, 如 `http://example.com` |     聊天、成书     |
  |    `"run_command"`    |   发送命令   |    command   |       String, 一条命令，包括`/`       | 聊天、成书、告示牌 |
  |    `"change_page"`    |   切换页码   |     page     |        Int, 页码        |        成书        |
  |  `"suggest_command"`  |   输入命令   |    command   |       String, 一条命令，包括`/`       |        聊天        |
  | `"copy_to_clipboard"` | 复制至剪贴板 |    value     |       String       |     聊天、成书     |

#### 悬浮事件(1.21.5+) ["hover_event"](https://zh.minecraft.wiki/w/?curid=49233#%E6%82%AC%E5%81%9C%E4%BA%8B%E4%BB%B6)
  |  **action**   |   描述   |            **附加标签**             |
  | :-----------: | :------: | :------------------------------: |
  |  `"show_text"`  | 显示文字 |     `{value:"任意文本组件"}`        |
  |  `"show_item"`  | 显示物品 |     `{id:"",count:,components:{}}`      |
  | `"show_entity"` | 显示实体 | `{uuid:"",id:"",name:""}` |

#### 点击事件(~1.21.4) ["clickEvent"](https://zh.minecraft.wiki/w/%E6%96%87%E6%9C%AC%E7%BB%84%E4%BB%B6?oldid=1015402#%E7%82%B9%E5%87%BB%E4%BA%8B%E4%BB%B6)
  |     **action**      |     描述     |     **value**      |       可用性       |
  | :-----------------: | :----------: | :----------------: | :----------------: |
  |     `"open_url"`      |   打开网页   | url, 如 `http://example.com` |     聊天、成书     |
  |    `"run_command"`    |   发送命令   |       String, 一条命令，包括`/`       | 聊天、成书、告示牌 |
  |    `"change_page"`    |   切换页码   |        Int         |        成书        |
  |  `"suggest_command"`  |   输入命令   |       String, 一条命令，包括`/`       |        聊天        |
  | `"copy_to_clipboard"` | 复制至剪贴板 |       String       |     聊天、成书     |

#### 悬浮事件(~1.21.4) ["hoverEvent"](https://zh.minecraft.wiki/w/%E6%96%87%E6%9C%AC%E7%BB%84%E4%BB%B6?oldid=1015402#%E6%82%AC%E5%81%9C%E4%BA%8B%E4%BB%B6)
  |  **action**   |   描述   |            **value** (1.16+已弃用)             |               content               |
  | :-----------: | :------: | :------------------------------: | :---------------------------------: |
  |  `"show_text"`  | 显示文字 |             JSON文本             |              JSON文本               |
  |  `"show_item"`  | 显示物品 |     `'{id:"",Count:,tag:{}}'` / `'{"id":"","count":"","components":{}}'`     |    `{"id":"","Count":"","tag":""}` / `{"id":"","count":"","components":{}}`    |
  | `"show_entity"` | 显示实体 | `'{type:"",id:"",name:"",tag:{}}'` | `{"name":JSON文本,"type":"","id":""}` |
